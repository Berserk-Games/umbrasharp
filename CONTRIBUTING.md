# LLMs

many projects take a "human-in-the-loop" standpoint, UmbraSharp does not. usage of LLMs in relation to this project is strictly prohibited

- PRs including LLM written code will be rejected
- any issue/PR written by an LLM will be closed and ignored
- any comment written by an LLM will be deleted and ignored
- if a commit written by an LLM that flies under the radar and is noticed later, it will be reverted however possible
